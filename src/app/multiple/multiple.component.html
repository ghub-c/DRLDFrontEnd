How does it work?


This interface provides the connection for single flights with our Deep Reinforcement Learning Python Backend.

Single flights mean a end-to-end route from point A to point B. You just have to select origin and destination numbers and those will be automatically translated into real coordinates at the map.

Trained weights are available, so there is no need to train the drone to learn a specific route if this was previously performed.

We also aim to provide training and testing data in order to compare the performance and efficiency for each of the DRL algorithms.
